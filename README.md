# Gerador de time pokémon aleatório
Em sua primeira versão será apenas na pokedéx do game Pokémon Fire Red

## Público-alvo
Jogadores de Pokémon fire red que querem um novo desafio. Ao usar esse software tu poderás gerar um time aleatório para um novo game
Tente zerar o game com um time aleatório

## Como rodar
```
python index.py
```
